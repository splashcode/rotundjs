Rotund.js

The javascript library for donuts style charting

Download (v0.1) View project on GitHub

Type 1 : one donut, multiples segments

var chart1 = new Rotund({
    container : 'chart1', 
    tooltip : {
        defaultText : 'Who <br/> ate <br/> donuts?', 
        color : '#FFF'
    },
    series : [{
        name : 'people',
        data : [{
            name : 'John',
            value : 6
        },{
            name : 'Jim',
            value : 4
        }]
    }]
});

Type 3 : multiples donuts, multiples segments by donut
